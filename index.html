Project is on the console


VS Code is an IDE/platform used to code. You can set it up by installing the app and going to the extensions tab and and typing in "Live Server." Next, you need to click on the extension that says live server and then click install.

The console is a screen which is used to see if any errors are occuring in the code. It is easy to use and open the console. To find any errors, you can print values of any kind of variable into the console by using the command "console.log([value]);" The console will tell tell you if an error has occured and in what line. To check the console, right click, click "inspect" and then click "console."
